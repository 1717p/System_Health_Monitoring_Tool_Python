# 💻 System Health Monitoring Tool (Python)

## 📌 Overview

A desktop-based system monitoring application built using Python to track real-time CPU, memory, and disk usage. The tool provides live system metrics with a simple graphical interface and alerts users when resource usage exceeds defined thresholds.

---

## 🚀 Features

* 📊 Real-time monitoring of CPU, Memory, and Disk usage
* 🔄 Automatic periodic updates of system metrics
* ⚠️ Threshold-based alerts with desktop notifications
* 🖥️ Interactive GUI built using Tkinter
* 🛠️ Debugged and tested for accurate and stable performance

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** psutil, tkinter, plyer
* **Concepts:** System Monitoring, Debugging, Event-driven updates

---


## 🔍 Key Learnings

* Working with system-level metrics using Python
* Implementing real-time data updates
* Designing GUI applications with Tkinter
* Handling alerts and notifications
* Debugging and testing for reliable execution
